## Hello

这里是小鑫，也可以叫我 Alexin。

我在整理一个更像自己的小站：记录生活里的小片段、照片里的风景，也把折腾博客、图床、服务器、页面和工具的过程写成可以回头看的笔记。

现在主要在做这些事：

- 给 `blog.alexin.cn` 写生活记录、相册和技能干货。
- 折腾 Typecho / Anheyu / 图床 / 音乐页，让博客更顺手。
- 把零散经验整理成文章模板和 Codex Skill。
- 用照片、文字和一点点代码，把日常保存得更清楚。
